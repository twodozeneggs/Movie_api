myFlix

web application will provide users with access to information about different movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies.

Business Logic URL HTTP Method Query parameters Request body data format Response body data format
Return a list of ALL movies to the user /movies GET None None A JSON object holding data about all movies
Return data about a single movie by title to the user /movies/[title] GET :title None A JSON object holding data about the selected movie containing description, genre, director and img URL. Example: 
             
   {
     Title: '', 
     Description: '',
     Year: '', 
     Genre: {
       name: '',
       description: ''
     } 
     Director: {
       name: '', 
       birth: '',
       death: ''
       bio: ''
     },
     Actors: {

     }
     ImagePath: ''
   }
Return data about a genre (description) by name/title (e.g., “Thriller”) /movies/genres/[genre] GET :genre None A JSON object holding data about the selected genre. Example: 
             
   { 
     name: 'Sci-fi', 
     description: '', 
   }
Return data about a director (bio, birth year, death year) by name /movies/directors/[name] GET :name None A JSON object holding data about the selected director. Example: 
             
   { 
     name: 'Peter Jackson', 
     birth: '1961', 
     death: '-',
     bio: ''
   }
Allow new users to register /users POST None A JSON object holding data about the user to add. Example: 
             
   { 
     "username": "Sam", 
     "email": "sami.sepehri@gmail.com", 
     "password": "St@rt2022!"
     "birthday": "17/06/1985", 
     "favorites": [
       "Interstellar", "Forrest gump"
     ] 
   }
A JSON object holding data about added user including ID. Example: 
             
   { 
     id: '1',
     "username": "Sam", 
     "email": "sami.sepehri@gmail.com", 
     "password": "St@rt2022!"
     "birthday": "17/06/1985", 
     "favorites": [
       "Interstellar", "Forrest gump"
     ] 
   }
Allow users to update their user info (username) /users/[username] PUT :username A JSON object holding the new username. Example: 
             
   { 
     "username": "Mary"
   }
A JSON object with the updated user profile. Example: 
             
   { 
     id: '1',
     username: 'Mary', 
     email: 'mary.rahmanpour@gmail.com', 
     password: 'ihatelemons'
     birthday: '12/09/1984', 
     favorites: [
       'Bee Movie', 'Parent Trap'
     ] 
   }
Allow users to add a movie to their list of favorites /users/[username]/[movie] POST :username, :movie None A text message indicating the movie was added
Allow users to remove a movie from their list of favorites /users/[username]/[movie] DELETE :username, :movie None A text message indicating the movie was deleted
Allow existing users to deregister /users/[username] DELETE :username None A message indicating the user was deregistered
